Talend (Pronunciation: TAL-end) is an open source software vendor that provides data integration, data management, enterprise application integration and big data software and services. Headquartered in Redwood City, California, Talend has offices in North America, Europe and Asia, and an international network of technical and service partners. Customers include eBay, Virgin Mobile, Sony Online Entertainment, Deutsche Post and Allianz.[citation needed] It has 400 employees in 14 offices in 7 countries
